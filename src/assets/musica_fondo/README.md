# Carpeta para la Música de Fondo 🎵💍

¡Hola Fernanda y Diego! He creado esta carpeta dedicada para que puedan subir la pista de audio que deseen usar como música de fondo de la tarjeta digital.

### Instrucciones para subir su música:
1. Utilicen el explorador de archivos del editor en el panel lateral.
2. Suban su pista en formato de audio (por ejemplo, `.mp3` o `.wav`) dentro de esta carpeta (`/src/assets/musica_fondo/`).
3. Pueden llamarla como gusten (por ejemplo, `cancion_boda.mp3`).

### Cómo reemplazarla en la invitación:
Si la suben con el nombre `cancion_boda.mp3`, me pueden avisar:
> *"Usa el archivo `cancion_boda.mp3` que subí en la carpeta de música de fondo"*

Y yo actualizaré la tarjeta de inmediato para que reproduzca su hermosa melodía elegida. 🎶✨
